# MCP Server (ActiveCampaign → Claude)

This is a lightweight MCP-style server exposing ActiveCampaign data.

## Features
- Automations
- Contacts
- Campaigns
- Emails

## Setup

1. Copy .env.example → .env
2. Add API credentials
3. Run:

npm install
npm start

## Endpoints

- /mcp/automations
- /mcp/contacts
- /mcp/campaigns
- /mcp/emails
